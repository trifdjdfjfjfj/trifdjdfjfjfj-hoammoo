# hoammo — static starter for chotainguyenmmo.wisp.uno

Deploy:
1. Create GitHub repo `trifdjdfjfjfj/trifdjdfjfjfj-hoammoo` (or push to existing).
2. Push files to `main` branch.
3. In repo Settings → Pages, set Source: `main` / root and save.
4. Ensure DNS: create a CNAME record for `chotainguyenmmo` -> `trifdjdfjfjfj.github.io`.
5. Wait for DNS propagation and for GitHub to provision HTTPS.
